📘 Introduction to Excel

Step 1 — What is Excel?

Microsoft Excel is a spreadsheet program to store, organize, and analyze data.

Step 2 — Understanding the Interface

  • Ribbon — Toolbar with tabs like Home, Insert, Page Layout
  • Workbook — File containing sheets
  • Worksheet — Single spreadsheet in a workbook
  • Cells — Intersection of row & column (e.g., A1)
  • Formula Bar — Where formulas are typed

🟡 SECTION 2 — INTERMEDIATE EXCEL

🧩 Formulas • Charts • PivotTables

🧩 Intermediate Formulas

Step 1 — Text Functions

  • =LEFT(A1,4) — First 4 chars
  • =RIGHT(A1,3) — Last 3 chars
  • =MID(A1,3,5) — From 3rd position

Step 2 — Logical

=IF(A2>100,"Pass","Fail")

Step 3 — Lookup

  • =VLOOKUP("Apple",A2:C10,2,FALSE)
  • =XLOOKUP("Apple",A2:A10,B2:B10)

🔵 SECTION 3 — ADVANCED EXCEL

🧠 Advanced Formulas

Step 1 — INDEX & MATCH

Step 2 — INDIRECT

Step 3 — OFFSET
