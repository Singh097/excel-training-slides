🟢 SECTION 1 — BASICS OF EXCEL

📘 Data Entry • Formatting • Essential Functions

📘 Introduction to Excel

  • What is Excel? — Spreadsheet software to store, organize & analyze data.
  • Workbook & Worksheet — Files & sheets.
  • Cells & Formula Bar — Where you type values and formulas.

Quick tip Ctrl + S Save frequently. (Windows)

✏️ Basic Functions & Formatting

  • Data entry: click cell → type → Enter
  • Formatting: fonts, borders, fill.
  • Number formats: Currency / Percentage / Date
Example — SUM / AVERAGE / MIN / MAX
Values in A1:A5 = [10, 20, 30, 40, 50]
=SUM(A1:A5) → 150
=AVERAGE(A1:A5) → 30
=MIN(A1:A5) → 10 · =MAX(A1:A5) → 50

🗂 Data Management Tools

  • Sort (A→Z / Z→A)
  • Filter (dropdowns)
  • Find & Replace (Ctrl + H)
  • Data Validation (dropdown lists)
Example — Data Validation Create a dropdown in cell B1 with items: Red, Green, Blue — choose from list.

🟡 SECTION 2 — INTERMEDIATE EXCEL

🧩 Formulas • Charts • PivotTables

🧩 Intermediate Formulas

  • Text functions — LEFT, RIGHT, MID
  • Logical — IF
  • Lookup — VLOOKUP, XLOOKUP
Text examples
=LEFT("Revenue",4) → "Reve"
=RIGHT("Revenue",3) → "nue"
=MID("Dataset",3,4) → "tase"
Logical example
With A2 = 120: =IF(A2>100,"Pass","Fail") → "Pass"
Lookup examples (small table)
Table A2:B4 — Apple:10, Banana:20, Cherry:30
=VLOOKUP("Banana",A2:B4,2,FALSE) → 20
=XLOOKUP("Cherry",A2:A4,B2:B4) → 30
=INDEX(B2:B4, MATCH("Apple",A2:A4,0)) → 10

📊 Charts & Visualization

  • Select data → Insert chart
  • Format titles, axes, labels
  • Use conditional formatting for quick visuals
Quick example Data: Sales Jan–May = [120, 150, 110, 180, 210] → Insert a column chart to show trend.

📈 PivotTables & PivotCharts

  • Insert → PivotTable
  • Drag fields → Rows / Columns / Values / Filters
  • Create PivotCharts for visualization
Example Using a sales table, put Product in Rows, Month in Columns and Sum of Sales in Values → get quick monthly totals by product.

🔵 SECTION 3 — ADVANCED EXCEL

🧠 Power Formulas • Macros • Automation

🧠 Advanced Formulas

  • INDEX & MATCH
  • INDIRECT & OFFSET (dynamic references)
  • Array formulas
INDEX + MATCH example
Table A2:A4 products = [A, B, C]; B2:B4 values = [100,200,300]
=INDEX(B2:B4, MATCH("C",A2:A4,0)) → 300
INDIRECT example
=INDIRECT("B"&5) reads cell B5 (useful for building dynamic references)

🤖 Macros & VBA Basics

  • Record macros to automate steps
  • Open VBA Editor: Alt + F11
  • Run macros to repeat tasks
Example — simple macro recorded Record: format header row → Save macro "FormatHeader" → Run later to apply same formatting.

🔍 Power Query & Data Analysis

  • Data → Get & Transform
  • Use Power Query Editor to clean & transform
  • Merge/Append queries from multiple sources
Example Import a CSV → Remove header rows, change column types → Close & Load into worksheet.

🟣 SECTION 4 — PRODUCTIVITY BOOSTERS

⚡ Shortcuts • Tricks • Data Cleaning

💡 Tips & Shortcuts

  • Copy/Paste — Ctrl+C/V
  • Undo — Ctrl+Z
  • Select All — Ctrl+A
Speed tricks
Alt + = inserts AutoSum. Try it on a column of numbers.

🧹 Data Cleaning

  • Remove blank rows
  • Remove duplicates (Data → Remove Duplicates)
  • Text to Columns
  • Trim spaces: =TRIM(A1)
TRIM example
Cell A1 = " hello " → =TRIM(A1) → "hello"
